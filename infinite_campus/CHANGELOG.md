# Changelog

## [1.0.0] - 2026-03-13

### Added
- Initial release
- Infinite Campus parent portal authentication
- Student data fetching (grades, assignments, attendance, schedule, courses, terms)
- WhatsApp notifications via CallMeBot (free)
- Web-based configuration dashboard with ingress support
- Real-time status monitoring
- Change detection with persistent cache
- Configurable notification types (grades, assignments, attendance, announcements)
- Daily summary messages
- Multi-student support
- Auto-start on boot option
- Configurable polling interval (5 min - 2 hours)
